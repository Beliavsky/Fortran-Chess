# Chess Piece Assets

This directory contains chess piece images for the Windows GUI.

## Active Set

`sashite-western`

- source: https://sashite.dev/chess-pieces/
- asset family: Western chess pieces
- file format: PNG
- size: 256x256
- license note: the source page describes these as a collection of public domain images released under CC0

## File Mapping

- `wk.png` white king
- `wq.png` white queen
- `wr.png` white rook
- `wb.png` white bishop
- `wn.png` white knight
- `wp.png` white pawn
- `bk.png` black king
- `bq.png` black queen
- `br.png` black rook
- `bb.png` black bishop
- `bn.png` black knight
- `bp.png` black pawn
